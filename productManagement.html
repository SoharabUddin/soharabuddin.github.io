Product Management App

An Android application for managing products, developed using Kotlin.

Project Overview

The Product Management App is an Android application that allows users to manage products. The app includes features like adding, editing, and deleting products, user authentication, live product search, and a navigation drawer menu.

Key Features

  • Add, Edit, and Delete Products using Retrofit
  • User Authentication (Login, Signup, Logout)
  • Profile View with Photo Upload
  • Live Product Search in RecyclerView
  • Drawer Navigation Menu
  • Alert Popup for Product Addition with Image Upload

Components

MainActivity.kt

Manages the main interface, displaying the list of products and handling user navigation.

  • onCreate(): Initializes views and sets up the navigation drawer.
  • setupRecyclerView(): Configures RecyclerView for displaying products.
  • onProductItemClick(): Handles product item click events for editing.

ProductAdapter.kt

Custom adapter for managing product data in a RecyclerView. Handles the binding of product information to view holders.

  • onCreateViewHolder(): Inflates the layout for product items.
  • onBindViewHolder(): Binds product data to the view holder.
  • deleteProduct(): Deletes a product from the list.

LoginActivity.kt

Manages user authentication, including login and signup functionalities.

  • onCreate(): Sets up the login and signup UI.
  • onLoginButtonClicked(): Authenticates user credentials.
  • onSignupButtonClicked(): Registers a new user.

ProfileActivity.kt

Displays and allows editing of user profile information, including photo upload.

  • onCreate(): Loads user profile data and sets up the photo upload functionality.
  • onSaveProfileClicked(): Saves updated profile information.

Technologies Used

  • Kotlin
  • Retrofit
  • RecyclerView
  • Navigation Drawer
  • Glide (for image loading)

Design Considerations

  • User Experience: Intuitive and seamless product management interface.
  • Data Handling: Efficient data retrieval and manipulation with Retrofit.
  • Security: Secure user authentication with Firebase.
